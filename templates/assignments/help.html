Help for Students

Viewing Your Assignments

If you are not signed in already, click 'Log In', in the top right of the page.
Then, enter your username and password given by your teacher.

You will then be taken to the home page, where you can view your assignments (if you have any).

{% load static %}

On this page, you can see the name, the class and the due date of each assignment. Click on an assignment to view further details.

Submitting Files

On the details page, each section of the assignment has a field to upload your files.
Click the browse button and select all the files you wish to upload.
When you are done, press the upload button at the very bottom of the page.

If you wish to delete any of the files you uploaded, a red trash can button can be clicked next to each file.

NOTE: If your submission contains, multiple folders, please upload it as a zip file.

Help for Teachers

Creating an Assignment

If you are not signed in already, click 'Log In', in the top right of the page.
Then, enter your username and password given by your administrator.

You will then be taken to the home page, where you can view any assignments you have assigned and a create button at the bottom. Press this go to the assignment creation page.

{% load static %}

Enter all the required fields, including:

  • The name of the assignment
  • The date and time the assignment is due
  • Which class you wish to give the assignment to
  • The details for the assignment. This box is a "rich text" editor, so you can include features from word processors such as bold/italic text and dot points

Then, click submit at the bottom.

{% load static %}

Adding Sections

After you have created an assignment, you will be taken to the details page where there is a button at the bottom to add a new section.
When this button is clicked, a form is added at the bottom with the required fields to add a new section. This functions similarly to the "create assignment" form.

{% load static %}

After filling out this form, you can press the create button to add the section to the assignment. If you don't want to add it, press the delete button.

If you wish to edit or delete a section after it has been added, a green edit button and a red delete button are avaiable next to each section to achieve this.

Viewing Submissions

At the top of each assignment details page, a "submissions" button can be clicked to view your students' submissions.

{% load static %}

Click on a different student name on the left to see their submission.
